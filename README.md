# ✈️ DreamScape – Travel & Tourism Website

DreamScape is a modern and responsive travel and tourism website designed to provide users with an engaging platform to explore destinations, discover travel packages, and plan their trips. The project focuses on creating an intuitive user experience with visually appealing layouts, responsive design, and seamless navigation.

---

## 🚀 Features

- Responsive and modern UI
- Interactive homepage
- Popular destination showcase
- Travel package listings
- Destination image gallery
- Booking interface
- Smooth navigation
- Mobile-friendly design
- Attractive animations and layouts

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/DreamScape.git

cd DreamScape
```

Open the project in your preferred code editor.

Run the application by opening:

```text
index.html
```

or use the **Live Server** extension in Visual Studio Code.
